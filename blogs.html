Question-1: What are the different ways to select an element in the DOM?

Answer: There are several ways to select an element in DOM. For Example:
1. selecting by ID.
-document.getElementById()
2. selecting by class Name.
-document.getElementsByClass()
3. selecting by tagName.
-document.getElementByTagName() 4. document.querySelector().
5. document.querySelectorAll().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer: 1. innerHTML
- This includes HTML tags, attributes and text.
- We can get or set HTML using JS. 2. innerText
- We can get or set the text content of an element. 3. textContent
- We can get or set the text content of an element.
